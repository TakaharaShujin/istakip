# İş takip sistemi

## Kurulum (geliştirmek için)
Github'dan proje indirilir:
```sh
git clone https://github.com/yasinaydin/istakip
```

Proje klasörünün içine girilir:
```sh
cd istakip
```

Gerekli NPM ve Bower modülleri kurulur:
```sh
npm install
bower install
```

Proje nodemon ile çalıştırılır:
```sh
nodemon
```
